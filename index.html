Let's have a look at how to dockerize an angular application.

Requirements :

  1. Node Installed in your system.
  2. Angular CLI installed on your system.
  3. Docker installed on your system
Let's Start

6. Now from the command line we will build our image using the command :
    docker build --rm -t myapp .

Here, 
  • docker build will build our image
  • --rm flag is used to delete the cached image or intermediate image created by docker
  • -t flag helps to tag our image
  • . specifies the dockerfile path.
After running this command, our image is created.
Verify it by using the command: docker images.

Now, we use docker run -d -p 81:80 
This runs our application in background on port 81.
